

## Acceso: utilización del punto (.) y las comillas dobles (")

Para acceder directamente a un elemento de un array de un determinado documento se
puede utilizar la siguiente sintaxis:

    "array.index"

Debemos tener en cuenta en los arrays en MongoDB **comienzan en el índice 0,** es decir, el primer elemento siempre tiene subíndice 0. 

Al acceder a un elemento concreto del array mediante su índice, es obligatorio **encerrar la clave completa entre comillas dobles** para que MongoDB pueda interpretarla correctamente. 

Si se indica un subíndice que no existe en el array, la operación no devolverá ningún resultado (en consultas) o no se realizará ninguna modificación (en actualizaciones).

_Por ejemplo, vamos a subir un punto la primera nota del alumno que estamos
_utilizando en todos los ejemplos (debemos utilizar la opearción sw actualización CRUD update()):

    >db.alumnos.**updateOne**( {nombre:"Abel"} , { $inc : { **"nota.0"** : 1 } } )

    > db.alumnos.find()  
    {  
        "_id" : ObjectId("56df11d778549bdfbf2125e3"),  
        "nombre" : "Abel",  
        "apellidos" : "Bernat Carrera",  
        "edad" : 22,  
        "dirección" : {  
            "calle" : "Mayor",  
            "numero" : 7,  
            "cp" : "12502"  
        },  
        "nota" : [  
            9.5,  
            7.5,  
            9  
        ]  
    }




## Consultas: $in, $nin, $all

**sin operador**{.azul}

Las consultas dentro de arrays de Mongo son muy sencillas.

La más sencilla es cuando buscamos un valor de un tipo sencillo, y en ese caso
lo que hará Mongo es buscar en todo el array por si está ese valor. Es decir,
exactamente igual que lo que hemos hecho hasta ahora.

        db.collection.find ( { clave_array : valor } )

Mirémoslo en un ejemplo. Vamos a crear dos documentos que tengan un array
cada uno, por ejemplo de colores. Lo creamos en una colección nueva, llamada
**colores** , en dos documentos con el mismo campo de tipo array, **color** ,
pero con datos diferentes:

    > db.colores.insert({color: ["rojo","azul","amarillo"]})  
    > db.colores.insert({color: ["negro","blanco","rojo"]})  
    > db.colores.find()
    { "_id" : ObjectId("56e1438ff6663c8169030e09"), "color" : [ "rojo", "azul", "amarillo" ] }  
    { "_id" : ObjectId("56e14398f6663c8169030e0a"), "color" : [ "negro", "blanco", "rojo" ] }

Como se ve en la sintaxis, elegir los documentos que tienen un campo (en este caso
de array) que contenga un valor, es igual de sencillo que cuando se trata de un
campo de tipo string, por ejemplo:

    > db.colores.find({color:"rojo"}) 

    { "_id" : ObjectId("56e1438ff6663c8169030e09"), "color" : [ "rojo", "azul", "amarillo" ] }  
    { "_id" : ObjectId("56e14398f6663c8169030e0a"), "color" : [ "negro", "blanco", "rojo" ] }

También se pueden utilizar **expresiones regulares** :

_Por ejemplo, buscar colores que contengan ro (como el **ro**jo , el neg**ro**):_

    > db.colores.find({color: /ro/ })  

    { "_id" : ObjectId("56e1438ff6663c8169030e09"), "color" : [ "rojo", "azul", "amarillo" ] }  
    { "_id" : ObjectId("56e14398f6663c8169030e0a"), "color" : [ " Siguiendo con la base de datos **cine**, realiza los siguientes ejercicios:negro", "blanco", "rojo" ] }

También podemos utilizar los siguientes operadores:

**$in**{.azul}

Busca documentos cuyo array contenga al menos uno de los valores indicados. **Equivale a un OR​**. La sitaxis es:

    clave : { $in : [valor1 , valor2 , ... , valorN] }

En el ejemplo, mostrará los documentos que contiene el color amarillo o el lila.

   > db.colores.find({color: {$in : ["amarillo","lila"]} })

    { "_id" : ObjectId("56e1438ff6663c8169030e09"), "color" : [ "rojo", "azul", "amarillo" ] } 
  
  **$nin**{.azul}

Busca documentos cuyo array no contenga ninguno de los valores indicados. Es lo contrario a $in. La sintaxis es:

    > db.colores.find({color: {$nin : ["negro"]} })

    { "_id" : ObjectId("56e1438ff6663c8169030e09"), "color" : [ "rojo", "azul", "amarillo" ] }

  **$ all**{.azul}

Busca documentos cuyo array contenga **todos** los valores indicados. **Equivale a un AND.**​

_Por ejemplo, vamos a buscar los documentos que tienen el color rojo y azul.
_
    > db.colores.find({color : { $all : ["rojo","azul"]} })  

    { "_id" : ObjectId("56e1438ff6663c8169030e09"), "color" : [ "rojo", "azul", "amarillo" ] }

  **Subíndice**{.azul}

Si queremos mirar exactamente una determinada posición del array, podemos
especificar la posición inmediatamente después de la clave, **separada por un
punto**. Recuerde que la primera posición es la **0**. Debemos poner **entre
comillas dobles la clave y la posición**, sino no sabrá encontrarla.

__Por ejemplo, buscamos los documentos que tienen el rojo en la primera posición.
_
    > db.colores.find({"color.0" : "rojo"} )  

    { "_id" : ObjectId("56e1438ff6663c8169030e09"), "color" : [ "rojo", "azul", "amarillo" ] }


!!! note "Nota"
    Acceder a una determinada posición es fácil, pero no es tan fácil acceder a una
    posición calculada, por ejemplo en la última posición. Ya hace falta conocimientos uno
    poco más avanzados de JavaScript, para poner dentro del **find()** una función en
    JavaScript, y actuar dentro de ésta.
---

  **$size**{.azul}

El operador **$size** nos servirá para hacer condiciones sobre el número
elementos de un array.

Por ejemplo, incorporamos 2 documentos nuevos, con 2 y 4 elementos respectivamente, para poder
comprobarlo:

    > db.colores.insertOne({color: ["negro","blanco"]})  

    > db.colores.insertOne({color: ["naranja","gris","lila","verde"]})  

    > db.colores.find()  

    { "_id" : ObjectId("56e1438ff6663c8169030e09"), "color" : [ "rojo", "azul", "amarillo" ] }  
    { "_id" : ObjectId("56e14398f6663c8169030e0a"), "color" : [ "negro", "blanco", "rojo" ] }  
    { "_id" : ObjectId("56e16972aa3c92aaed389da6"), "color" : [ "negro", "blanco" ] }  
    { "_id" : ObjectId("56e16990aa3c92aaed389da7"), "color" : [ "naranja", "gris", "lila", "verde" ] }

Ahora vamos a seleccionar los documentos que tienen 4 colores

    > db.colores.find({color:{$size:4}})  

    { "_id" : ObjectId("56e16990aa3c92aaed389da7"), "color" : [ "naranja", "gris", "lila", "verde" ] }

!!! note "Nota"
    El operador **$size** sólo admite un valor numérico, y no se pueden concatenar
    expresiones con otros operadores, como por ejemplo intentar la condición de que el
    tamaño del array sea menor o igual a un determinado valor. 

  **$slice**{.azul}

El operador **$slice** no es un operador que pueda ponerse en las condiciones
(criterios), sino que servirá para extraer determinados elementos del array,
por el número de orden de estos elementos en el array. Sólo podremos ponerlo, pero
tanto, en el segundo parámetro del **find()**.

La sintaxis es:

    clave : {$slice : x }

Los valores que puede tomar **x** son:

  * Números positivos: será el número de elementos del principio (por la izquierda)
  * Números negativos: será el número de elementos del final (por la derecha)
  * Un array de 2 elementos (**[x,y]**): mostrará a partir de la posición **x** (0 es el primero), tantos elementos como indique **y**

Por ejemplo, vamos a mostrar los dos primeros colores de cada documento:

    > db.colores.find({} , {color:{$slice:2} })  

    { "_id" : ObjectId("56e1438ff6663c8169030e09"), "color" : [ "rojo", "azul" ] }  
    { "_id" : ObjectId("56e14398f6663c8169030e0a"), "color" : [ "negro", "blanco" ] }  
    { "_id" : ObjectId("56e16972aa3c92aaed389da6"), "color" : [ "negro", "blanco" ] }  
    { "_id" : ObjectId("56e16990aa3c92aaed389da7"), "color" : [ "naranja", "gris" ] }

O mostrar el último color:

    > db.colores.find({} , {color:{$slice:-1 }})  

    { "_id" : ObjectId("56e1438ff6663c8169030e09"), "color" : [ "amarillo" ] }  
    { "_id" : ObjectId("56e14398f6663c8169030e0a"), "color" : [ "rojo" ] }  
    { "_id" : ObjectId("56e16972aa3c92aaed389da6"), "color" : [ "blanco" ] }  
    { "_id" : ObjectId("56e16990aa3c92aaed389da7"), "color" : [ "verde" ] }

O mostrar el tercer elemento, tengan los que tengan. Recuerde que el segundo
elemento, es el de la posición 2, y queremos mostrar 1.

    > db.colores.find({} , {color:{$slice:[2,1] }})  

    { "_id" : ObjectId("56e1438ff6663c8169030e09"), "color" : [ "amarillo" ] }  
    { "_id" : ObjectId("56e14398f6663c8169030e0a"), "color" : [ "rojo" ] }  
    { "_id" : ObjectId("56e16972aa3c92aaed389da6"), "color" : [ ] }  
    { "_id" : ObjectId("56e16990aa3c92aaed389da7"), "color" : [ "lila" ] }



## Actualización: $ y $[]

**$**{.azul}

Identifica un elemento en un array para actualizar sin especificar explícitamente la posición del elemento en un array.


Ejemplo: Modificar la primera nota mayor a 8 en el array nota 
 
    db.alumnos.updateOne(
      { "_id": ObjectId("56df11d778549bdfbf2125e3"), "nota": { $gt: 8 } },
      { $set: { "nota.$": 10 } }
  )

**$[]**{.azul}

El operador $[] se utiliza para actualizar todos los elementos en un array.

Ejemplo: Aumentar en 1 punto todas las notas en el array nota

    db.alumnos.updateOne(
        { "_id": ObjectId("56df11d778549bdfbf2125e3") },
        { $inc: { "nota.$[]": 1 } }
    )



## Inserción: $push 

La forma más sencilla de introducir un elemento en un array es utilizar
**$push** sin más. Si existía el array, introducirá el o los nuevos elementos en el
final. Si no existía el array, lo creará con éste o estos elementos.

La sintaxis es:

    { $push : { clave : elemento } }

Por ejemplo vamos a añadir una nota al alumno de siempre, y pongámosla diferente
para ver que se introduce al final:

    > db.alumnos.updateOne( {nombre:"Abel"} , { $push : { nota : 7 } } )  


    > db.alumnos.find()  
    {  
        "_id" : ObjectId("56df11d778549bdfbf2125e3"),  
        "nombre" : "Abel",  
        "apellidos" : "Bernat Carrera",  
        "edad" : 22,  
        "dirección" : {  
            "calle" : "Mayor",  
            "numero" : 7,  
            "cp" : "12502"  
        },  
        "nota" : [  
            9.5,  
            7.5,

    }

## Eliminación: $pop y $pull {.azul}

Hay más de una forma de eliminar elementos de un array.

**$pop**{.azul}

**$pop**. La sintaxis es

    { $pop : { clave : posición } }

Donde en posición podremos poner:

  * -1 , y borrará el primer elemento
  * 1 , y borrará el último

primero.

**$pull**{.azul}

Con este modificador borraremos los elementos del array que coincidan
con una condición, estén en la posición que estemos. Observe cómo se puede
eliminar más de un elemento.

Para poder comprobarlo bien, primero insertamos otro elemento al final de
el array, con el valor **7.5** (si ha seguido los mismos ejemplos que en
estos apuntes, ese valor ya se encuentra en la segunda posición).

    > db.alumnos.updateOne( {nombre:"Abel"} , { $push : { nota : 7.5 } } )  
    
     > db.alumnos.find()

        "_id" : ObjectId("56debe3017bf4ed437dc77c8"),
